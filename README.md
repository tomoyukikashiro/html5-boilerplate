## Build
### Development
Execute default task.

* command
  ```shell
  grunt
  ```

* watch sass
  ```shell
  grunt watch:sass
  ```

* watch js
  ```shell
  grunt watch:js
  ```

### Production
Execute prod task.

* command

  ```shell
  grunt prod
  ```
